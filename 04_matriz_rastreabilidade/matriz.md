# MATRIZ DE RASTREABILIDADE  
## Sistema SRL

---

## Objetivo

A matriz de rastreabilidade tem como objetivo garantir que todos os requisitos funcionais do sistema estejam cobertos por pelo menos um caso de teste.

Isso permite identificar lacunas e assegurar cobertura mínima exigida.

---

## Requisitos Funcionais

RF-01: Permitir cadastro de usuário  
RF-02: Permitir autenticação  
RF-03: Permitir cadastro de laboratório  
RF-04: Permitir consulta de laboratórios  
RF-05: Permitir reserva de laboratório  
RF-06: Permitir cancelamento de reserva  

---

---

## Conclusão

Todos os requisitos possuem pelo menos um caso de teste associado, garantindo cobertura funcional mínima conforme exigido na atividade.
